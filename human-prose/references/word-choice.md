# Word Choice

词汇层面的 AI 写作模式。核心问题：用花哨的词替代准确的普通词。

## "Quietly" 和魔法副词

用"quietly"、"deeply"、"fundamentally"、"remarkably"、"arguably"给平淡描述注入虚假分量。

问题：读者感受到的是空洞，不是分量。

坏例子：
- "quietly orchestrating workflows"
- "a quiet intelligence behind it"
- "deeply transformative approach"

修正方式：删掉副词，看句子是否仍成立。如果成立，副词就是多余的。如果不成立，换成具体描述（数字、场景、对比）来传递你想要的分量。

## "Delve" 家族

AI 高频词表：delve, certainly, utilize, leverage (动词), robust, streamline, harness, foster, pivotal, multifaceted, comprehensive, cutting-edge, groundbreaking。

问题：这些词在 AI 输出中出现频率远高于人类写作，已成为 AI 文本的指纹。

修正方式：

| AI 词 | 换成 |
| --- | --- |
| delve into | look at / examine / dig into |
| utilize | use |
| leverage (v.) | use / build on |
| robust | strong / solid / reliable |
| streamline | simplify / speed up |
| harness | use / apply |
| foster | encourage / support |
| pivotal | important / key |
| comprehensive | full / complete / thorough |
| cutting-edge | new / recent / latest |

## "Tapestry" 和 "Landscape"

用华丽名词替代简单名词：tapestry, landscape, paradigm, synergy, ecosystem, framework, realm, journey。

问题：这些词模糊了具体含义，让读者无法形成画面。

坏例子：
- "The rich tapestry of human experience"
- "Navigating the complex landscape of modern AI"
- "The ever-evolving landscape of technology"

修正方式：问自己"具体是什么？"。如果是"AI 工具市场"，就写"AI 工具市场"。如果是"几十种互相竞争的框架"，就写"几十种互相竞争的框架"。

## "Serves As" 回避症

用"serves as"、"stands as"、"marks"、"represents"替代简单的"是"。

问题：AI 的重复惩罚机制让它回避基本系动词，导致句子变得做作。

坏例子：
- "The building serves as a reminder of the city's heritage."
- "The station marks a pivotal moment in the evolution of regional transit."

修正方式：直接用"是"。"这栋楼是城市遗产的一部分"比"这栋楼充当了城市遗产的提醒"更清楚。

## 自查规则

1. 搜索上面列出的 AI 高频词，逐个替换
2. 搜索副词（-ly 结尾），删掉不影响意思的
3. 搜索"serves as / stands as / marks / represents"，能换"是"就换
4. 如果一个词删掉后需要用更多字来补偿，说明它是有用的——保留它
